<div>we are also using Andor Neo cameras and they work pretty great. </div>
<div><br>
</div>
<div>Currently, we are refactoring the OpenSPIM plugin, so it can be used with a normal installation of Fiji and MicroManager, so our special package is not needed anymore. I have a recent nightly build available, based on MicroManager 1.4.18, which you can
 find at <a href="https://idisk-srv1.mpi-cbg.de/~gunther/OpenSPIM/mFiji.zip" target="_blank">https://idisk-srv1.mpi-cbg.de/~gunther/OpenSPIM/mFiji.zip</a> The corresponding OpenSPIM plugin is available at <a href="https://idisk-srv1.mpi-cbg.de/~gunther/OpenSPIM/SPIMacquisition.jar" target="_blank">https://idisk-srv1.mpi-cbg.de/~gunther/OpenSPIM/SPIMacquisition.jar</a> and
 has to be put into the mmplugins/ directory. This version also works on 64bit machines.</div>
<div><br>
</div>
<div>With this version, you can just create your setup as usual, just select the Andor SDK3 as camera.</div>
<div><br>
