<div>Hi Ann,</div>
<div><br>
</div>
<div>The sample chamber has an stl format on the openspim website, which is the normal file for most 3D printers. You won't be able to edit this file in autocad though. If you want to edit it before printing it, you'll have to use an online stl to .dwg converter
 to open it in autocad. Then, after editing, you'll have to re-export it as an stl file again for printing.</div>
<div><br>
</div>
<div>If you just want to test print it, the provided stl file will do fine.</div>
<div><br>
