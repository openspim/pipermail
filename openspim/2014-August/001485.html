Hi Britta, and in particular EMBO course participants using the ultramicroscope,

said microscope's software puts brackets into file names (what's next? I
fully expect somebody to put in musical notes, animals and little babies
into file names within the next years).

In addition to looking awkward and making for very long file names, this
also triggers a design flaw in ImageJ 1.x' macro language: closing
brackets are misinterpreted to indicate the end of macro parameters, and
consequently, say, a recorded File>Import>Image Sequence... will fail to
open anything.

To work around this issue, Britta suggested to write a macro that lets you
pick the directory containing the problematic files and rename them. This
macro exists now, in File>Fix Funny Filenames (simply update and restart
your Fiji).

If you find that the macro is incomplete, or wrong, or if you just want to
inspect the code for your education, reading pleasure, or to change it to
do something slightly different, feel free to hold down the Shift key
before clicking on the menu item "Fix Funny Filenames"; this will open the
macro in the script editor instead of executing it.

And of course, if you make changes to the macro, please feel free to share
the modified version here; that way we will keep the incredible spirit
felt at the EMBO course alive!
