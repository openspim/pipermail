Hi All,

We're planning on releasing 1.4.15 on Monday, September 24.

Since it has been 6 months since our last release (1.4.14), there have
been quite a few new added features, device adapters, and bugfixes. A
list of these will be posted here before or at the time of the
release.

Developers with SVN commit access:

We have not had a formal release schedule/process in the past
(something we might think about in the future), but it is very true
that vastly more people use the release versions, compared to the
nightly builds. If you are thinking of introducing significant new
functionality, it will be best to commit such changes after the
release date, so that advanced users who use the nightly builds will
have a chance to test the new functionality before it goes into the
next release version.

(We feel that 6 months is too long a period between releases, as we
would like to get new features and fixes to users more quickly. We are
considering introducing a semi-regular schedule (quarterly?
bimonthly?) for future releases.)

Best,
The Micro-Manager Team
