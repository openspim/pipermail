Hi Michael,

The error suggests that the configuration file is trying to load the 
COM1 device twice.

Do you have multiple serial devices in the configuration? The log seems 
to suggest not. Have you tried creating a configuration from scratch? 
It's possible there have been changes to SerialManager that require 
creating a new configuration file.

If these ideas don't help, could you please send an e-mail to the list 
containing your configuration file and the core log as attachments?

Thanks!
Luke
