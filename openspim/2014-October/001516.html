Oh, Yes. we solved the problem, as the suggestion http://openspim.org/pipermail/openspim/2014-May/000408.html

Since the Stradus device adapter is "secret", my ability to help is limited.
My suggestion is:
- First, please make sure you are using the Stradus device adapter
(mmgr_dal_Stradus.dll) from the 5 February 2014 nightly build of
Micro-Manager[1].
- Run OpenSPIM, but before clicking "Acquire SPIM Image", create a pixel
size:
- On Micro-Manager, click "Tools -> Pixel Size Calibration..."
- Click "New"
- Set Label to "Uncalibrated"
- Set Pixel Size (um/pixel) to 1.00
- Across from "Core - Initialize" click "Use" checkbox.
- Click "OK".
- Try "Acquire SPIM Image" again.

-Zuomin
