Hi Peter,

I just performed the setup as explained in the wiki. Yes though, we're
using the 50mW coherent 488 cube and with the options set it works ok.
I haven't seen that error and im on vacation at the moment so can't check
the setup directly.

I have to make mention that the picard software that picard supplies
doesn't drive the stage at all, however openspim micromanager does. I'm not
sure if others have encountered this with win7/64.

One final thing, on drivers,coherent still doesnt have a 64bit usb driver
for the cube series. So you have no choice but to use serial to control the
laser rather than usb (might be worth adding to the wiki).

Cheers Kieran
