Hi, All, 
    Sorry to disturb you again. I solved the error I reported last time. It's nothing to do with the laser property name :Workinghours . The real error to terminate the software is : java.lang.IllegalArgumentException: invalid range properties.
    It maybe the number of twister of the PicardStage is beyond the SPIM software can hold. After I set the number to zero with the PicardStage control software, the SPIM software can open.
    Sorry again!
