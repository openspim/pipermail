Hi All,

We have just released Micro-Manager 1.4.22.
This is a maintenance release for the main application, with bugfixes
and minor enhancements. There are also enhancements for various
devices and some new device adapters.

It can be downloaded (for Windows and Mac) from
http://micro-manager.org/wiki/Download_Micro-Manager_Latest_Release
where you can also view a list of changes since 1.4.21.

Thank you all for your contributions and feedback. It's what keeps
Micro-Manager going, so please keep it coming! If you find any issues
in version 1.4.22, please report them by starting a new thread on this
mailing list or sending a Problem Report (from Micro-Manager's Help
menu).

This will most likely be the last release version published by the
UCSF Micro-Manager Team. As previously announced on this mailing list
and elsewhere, Micro-Manager development will be continued by Open
Imaging (https://open-imaging.com/) starting in October. Open Imaging
will be providing future release version binaries as free downloads.

If you are wondering about Micro-Manager 2.0: the plan is to have a
beta version and then an official 2.0.0 release later this year, if
things go smoothly. After that point in time, 1.4.x will be maintained
in parallel for at least a year, perhaps longer.

Best regards,
The Micro-Manager Team
