Hi All,

We have just released Micro-Manager 1.4.21.
This is primarily a maintenance release with a lot of bug fixes. There
are also enhancements for various devices.

It can be downloaded (for Windows and Mac) from
http://micro-manager.org/wiki/Download_Micro-Manager_Latest_Release
where you can also view a list of changes since 1.4.20.

Thank you all for your contributions and feedback. It's what keeps
Micro-Manager going, so please keep it coming! If you find any issues
in version 1.4.21, please report them by starting a new thread on this
mailing list or sending a Problem Report (from Micro-Manager's Help
menu).

Unless special circumstances arise, our next release will be at the
end of July, 2015.

Best Regards,
