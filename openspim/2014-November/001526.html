Dear Frédéric,
in our open spim setup we had the same issue with the 488 laser like you. We detected laser light in the emission path. The solution - as already mentioned by Pete - is a clean up filter. We found a relative cheap filter from Edmund optics Part No.#65-086. This filter has only a blocking efficiency of 10^-4. However, the filter works fine in our GFP application. This filter fits nicely into the aperture in front of the slit, so there in no need for an additional holder.
