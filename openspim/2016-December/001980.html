Hi Matt,

We use a 488/561 Versalase with an ESio TTL controller to enhance
switching speed.  Our setup largely followed a design that was generously
provided by Johannes Girstmair in the Telford lab at University College
London, with a few modifications to fit our needs.  He provided me
extensive help that I can pass along if he has not done already.

Overall it works quite well.  If you use a TTL controller, you will want
to add the following to the Configuration Settings window in
Micro-Manager:

Laser_[A/B/C/D]_LaserEmission
Laser_[color]_DigitalPeakPowerSetting
TTL_State (I rename this setting 'color' and name the TTL states that
result in the desired outputs 'green', 'red' and 'off'.  This lets me set
multi-channel images by assigning the 'color' variable in Multi-D
Acquisition)

Turn on DigitalModulation in the Device Property Browser and you should be
set.  You have to switch LaserEmission on each time you boot up MM and if
you use a TTL controller PeakPower will determine the laser intensity.

Blanking mode will theoretically help acquire faster with minimized light
exposure.  However I have not got that to work; Micro-Manager's
auto-shutter seems to do the job well enough.

All the best,
