  Hello Monika,


we’re currently working on making OpenSPIM into a simple MicroManager plugin so the whole custom MicroManager package is not needed anymore. 
This will also solve your 64bit troubles. For the moment, I can provide you with a 64bit build based on MicroManager 1.4.18, which you can find at https://idisk-srv1.mpi-cbg.de/~gunther/OpenSPIM/mFiji.zip



You’ll also need the OpenSPIM plugin, which you can find at https://idisk-srv1.mpi-cbg.de/~gunther/OpenSPIM/SPIMacquisition.jar
 - this has to be put in mmplugins/
Please also make sure you have the 1.8 version of the Java JRE installed.


Let me know if that worked for you :-)
