Hello WIKIZONG, 

I'm one of the software developers on the OpenSPIM project; the good news is that the Orca Flash4 is successfully being used for the OpenSPIM project, in Heidelberg -- see http://openspim.org/Who_has_an_OpenSPIM%3F . 


We have also encountered this error message with Hamamatsu cameras before, usually because Fiji is taking up too much of the system's memory. Please try limiting Fiji's memory (Edit > Options > Memory & Threads...) to 1 GB or so. Additionally, this error message sometimes appears only on the first startup of Micro-Manager after booting the computer -- please try restarting Micro-Manager and see if the error message still appears. 


Hope this helps, 
Luke Stuyvenberg 


P.S. The OpenSPIM project has a mailing list -- openspim at openspim.org -- please feel free to contact me there if you have any more trouble! :-) Additionally, the Heidelberg lab is subscribed to that mailing list, so they might be able to provide more direct and useful advice if the Flash4 is still problematic for you.
